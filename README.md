# RimFix Constructor

Веб-конфигуратор автомобильных дисков. Позволяет выбрать тип кузова автомобиля, подобрать дизайн колёсных дисков и цвет их покраски, а затем автоматически наложить диски на изображение кузова с помощью OpenCV.js.

![RimFix Constructor](car_types/sedan.png)

## Возможности

- **8 типов кузовов** на выбор: седан, хэтчбек, купе, кабриолет, кроссовер, минивэн, пикап, универсал.
- **Библиотека дисков** — выбор дизайна колёсных дисков из встроенного набора.
- **Автоматическое определение положения колёс** с помощью OpenCV.js (template matching по шаблонному изображению).
- **Покраска дисков** — 18预设-цветов палитры (серебро, антрацит, чёрный глянец, бронза, золото, медь, розовое золото, титан и др.) плюс выбор собственного цвета через color picker.
- **Оригинальный цвет** — возможность оставить диск без покраски.
- **Адаптивный интерфейс** — отдельные раскладки для десктопа (боковая панель) и мобильных устройств (нижние sheet-панели с табами).
- **Тёмная тема** оформления.

## Используемые технологии

- **HTML / CSS / JavaScript** — интерфейс, логика и обработка изображений (всё в одном файле `wheel_configurator.html`).
- **OpenCV.js** — компьютерное зрение: поиск колёсных арок на изображении кузова через `cv.matchTemplate`.
- **Python** — простой локальный HTTP-сервер (`run_server.py`) для раздачи статики.

## Как это работает

1. Пользователь выбирает тип кузова — соответствующее изображение из `car_types/` отображается на canvas.
2. Пользователь выбирает дизайн дисков из `wheel_types/`.
3. OpenCV.js выполняет template matching: изображение-шаблон (`other_assets/replace_part.png`) ищется на изображении кузова, определяя позиции колёсных арок.
4. Выбранный диск масштабируется по размеру арки и накладывается на найденные позиции.
5. Если выбран цвет покраски — пиксели диска перекрашиваются с сохранением яркости (luminance), что сохраняет объём и рельеф диска.
6. Результат отображается на canvas в реальном времени.

## Установка и запуск

### Требования

- **Python 3.8+** — для запуска локального сервера.
- **Современный веб-браузер** с поддержкой Canvas и WebAssembly (для OpenCV.js).

### Шаги

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/Siririn77/rimfix-constructor.git
   cd rimfix-constructor
   ```

2. Запустите локальный сервер:

   ```bash
   python3 run_server.py
   ```

3. Откройте в браузере:

   ```
   http://localhost:8000/wheel_configurator.html
   ```

4. Выберите кузов, диски и цвет — результат появится на canvas.

> Сервер раздаёт статику на порту `8000`, отправляет CORS-заголовки и отключает кэширование (удобно при разработке).

## Использование

1. В боковой панели (на десктопе) или через нижние табы (на мобильном) выберите **тип кузова**.
2. Выберите **дизайн дисков** — диски автоматически найдут колёсные арки и наложатся на изображение.
3. Выберите **цвет покраски** диска из палитры или задайте свой цвет через color picker.
4. Для возврата к оригинальному цвету диска выберите «Оригинал» в палитре.

## Добавление новых типов кузовов и дисков

Типы кузовов и дисков задаются в JavaScript-массивах внутри `wheel_configurator.html`:

```javascript
const CAR_TYPES = [
    { id: 'sedan', name: 'Седан', src: 'car_types/sedan.png' },
    // ...добавьте сюда новый объект
];

const WHEEL_TYPES = [
    { id: 'wheel1', name: 'Диск 1', src: 'wheel_types/wheel1.png' },
    // ...добавьте сюда новый объект
];
```

Чтобы добавить новый диск:

1. Поместите изображение PNG (с прозрачным фоном) в папку `wheel_types/` (например, `wheel3.png`).
2. Добавьте запись в массив `WHEEL_TYPES`:

   ```javascript
   { id: 'wheel3', name: 'Диск 3', src: 'wheel_types/wheel3.png' }
   ```

3. Обновите страницу — новый диск появится в списке.

Аналогично для новых типов кузовов — изображение в `car_types/` + запись в `CAR_TYPES`.

> Рекомендуется использовать квадратные изображения дисков с прозрачным фоном и одинаковым разрешением.

## Разработка

Вся логика приложения — интерфейс, обработка изображений и покраска — находится в одном файле `wheel_configurator.html`. Для внесения изменений редактируйте этот файл. После изменений достаточно обновить страницу в браузере (кэширование отключено в `run_server.py`).

### Ключевые части кода

- `CAR_TYPES`, `WHEEL_TYPES`, `PALETTE` — данные кузовов, дисков и цветов.
- `loadOpenCV()` — динамическая загрузка OpenCV.js с автоматическим повтором при сбое.
- `applyWheelsWithOpenCV()` — поиск колёсных арок и наложение дисков.
- `getTintedWheelCanvas()` — покраска диска с кэшированием результата.
- `selectCar()`, `selectWheel()` — обработчики выбора.
